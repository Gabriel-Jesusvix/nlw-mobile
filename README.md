## Rodando localmente

Clone o projeto

```bash
  git clone https://github.com/Gabriel-Jesusvix/nlw-mobile
```

Entre no diretório do projeto

```bash
  cd mobile
```

Instale as dependências

```bash
  npm install
```

Inicie o servidor

```bash
  npm run start
```
```bash
Change local network baseURL (.env.example)
```
